📸 PhotoShare

PhotoShare est une application web développée avec Django qui permet aux utilisateurs de partager des photos et d’interagir entre eux.

🔹 Fonctionnalités
Inscription et connexion des utilisateurs
Publication de photos avec description
Fil d’actualité (feed)
Système de likes ❤️
Système de commentaires 💬
Suppression de ses propres photos
🛠️ Technologies
Python / Django
HTML / CSS
MySQL (ou SQLite)
📁 Structure
accounts/ : gestion des utilisateurs
photos/ : photos, likes, commentaires
templates/ : pages HTML
media/ : images uploadées
▶️ Lancer le projet
python manage.py runserver

Puis ouvrir :
http://127.0.0.1:8000/
